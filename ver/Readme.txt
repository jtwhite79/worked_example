Readme:

Listing of files included in the data release for "Towards reproducible environmental modeling for decision support: a worked example" by White and others, 2020.

In the pred directory are the MODFLOW-2005 input files supplied by the Edwards Aquifer Authority for the history-matching simulation.

Directories:

ref

Files:

eaa_ver.nam - the MODFLOW-2005 nam file
eaa_ver.wel - the MODFLOW-2005 well package file
eaa_ver.rch - the MODFLOW-2005 recharge package file
eaa_ver.oc - the MODFLOW-2005 output control file
eaa_ver.lpf - the MODFLOW-2005 layer properties file
eaa_ver.hfb - the MODFLOW-2005 hydraulic flow barrier file
eaa_ver.gmg - the MODFLOW-2005 geometric multigrid solver file
eaa_ver.drn - the MODFLOW-2005 drain package file
eaa_ver.dis - the MODFLOW-2005 discretization package file
eaa_ver.bas - the MODFLOW-2005 basic package file
