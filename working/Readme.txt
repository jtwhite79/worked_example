Readme:

Listing of files included in the data release for "Towards reproducible environmental modeling for decision support: a worked example" by White and others, 2020.

In the working directory is the main driver script, python frameworks and aux supporting files

Directories:

_data - aux supporting files
bin - pre-compiled binaries for windows, mac, and linux (ubuntu 18.04) of both MODFLOW-2005 and PESTPP-IES.  
flopy - the FloPy python framework used for the analyses
pyemu - the pyEMU python framework usef for the analyses


Files:

eaa.py - the driver script that implements all steps in the analyses
