Readme:

Listing of files included in the data release for "Towards reproducible environmental modeling for decision support: a worked example" by White and others, 2020.

In the pred directory are the MODFLOW-2005 input files supplied by the Edwards Aquifer Authority for the scenario hindcast simulation.

Directories:

ref

Files:

eaa_pred.nam - the MODFLOW-2005 nam file
eaa_pred.wel - the MODFLOW-2005 well package file
eaa_pred.rch - the MODFLOW-2005 recharge package file
eaa_pred.oc - the MODFLOW-2005 output control file
eaa_pred.lpf - the MODFLOW-2005 layer properties file
eaa_ver.hfb - the MODFLOW-2005 hydraulic flow barrier file
eaa_pred.gmg - the MODFLOW-2005 geometric multigrid solver file
eaa_pred.drn - the MODFLOW-2005 drain package file
eaa_pred.dis - the MODFLOW-2005 discretization package file
eaa_pred.bas - the MODFLOW-2005 basic package file
