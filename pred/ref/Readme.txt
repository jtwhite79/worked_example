Readme:

Listing of files included in the data release for "Towards reproducible environmental modeling for decision support: a worked example" by White and others, 2020.

In the pred/ref directory are the MODFLOW-2005 external array and list files

Files:

WEL_XXXX.dat - the well package list-style input data for stress period XXXX
DRN_XXXX.dat - the drain package list-style input data for stress period XXXX
rech_X.ref - the recharge package recharge array for stress period X
botm_layer_1.ref - the bottom elevation array for layer 1
hani_layer_1.ref - the horizontal hk anisotropy array for layer 1
model_top.ref - the model top elevation array
hk_layer_1.ref - the horizonatal hydraulic conductivity array for layer 1
ss_layer_1.ref - the specific storage array for layer 1
sy_layer_1.ref - the specific yield array for layer 1
strt_layer_1.ref - the initial groundwate level array for layer 1
vka_layer_1.ref - the vertical hydraulic conductivity array for layer 1
ibound_layer_1.ref - the active domain boundary array for layer 1
delr.ref - the column spacing array
delc.ref - the row spacing array
